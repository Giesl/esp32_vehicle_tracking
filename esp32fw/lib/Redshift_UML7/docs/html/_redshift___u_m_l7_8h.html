Detailed Description

Header file for the Redshift_UML7 class, which provides an interface to the UM7 sensor.

This file contains the definitions, constants, and declarations required to interact with the UM7 sensor. The UM7 sensor is a high-performance orientation sensor that provides data such as gyroscope, accelerometer, magnetometer, GPS, and orientation (yaw, pitch, roll). The Redshift_UML7 class facilitates communication with the sensor over a serial interface and provides methods to retrieve and process sensor data.
